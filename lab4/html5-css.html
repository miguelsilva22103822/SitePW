HTML5 e CSS

HTML5

HTML5 é a versão mais recente da linguagem HTML que é usada para formatar conteúdo na web.

Elemento Usei/Não usei Usado para
div 😀 Enquadramento para o poema
span 🙄
header 😀 Cabeçalho em todas as páginas
nav 😀 Para o menu de navegação
main 😀 Em todas as páginas para indicar o conteudo principal
article 🙄
section 😀 Para agrupar os elementos das páginas
aside 🙄
footer 😀 Rodapé em todas as páginas
details 😀 Em todas as imagens, uma descrição curta da imagem
meter 🙄
progress 🙄
abbr 🙄
address 🙄

CSS

CSS é uma linguagem usada para definir a aparência e o layout de elementos em uma página web, permitindo que os desenvolvedores criem sites visualmente apelativos e facilitanto a sua manutenção.

Seletor Usei/Não usei Usado para
universal 😀 Alterar a font em todo o site
tipo 😀 Estilizar vários elementos
classe 😀 Alterar certos elementos apenas
id 😀 Estilizar elementos únicos
atributo 🙄
pseudo-classe 😀 Cor dos hyperlinks
pseudo-elemento 😀 Cor da primeira letra do elemento h1
Combinadores de Seletores Usei/Não usei Usado para
_ 🙄
> 🙄
~ 🙄
+ 🙄
, 🙄
